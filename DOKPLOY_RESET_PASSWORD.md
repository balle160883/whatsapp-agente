# Resetear Contraseña en Dokploy

## Pasos para resetear la contraseña de tu cuenta:

1. Ve a tu aplicación en **Dokploy**
2. Haz clic en **Terminal** (el botón con el ícono de terminal)
3. Dentro de la terminal, ejecuta:
   ```bash
   npm run reset-password
   ```
